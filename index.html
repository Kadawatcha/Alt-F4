Le petit plus pour vos serveurs

Nous vous proposons ce que peu d'autres bots vous proposent :
Vous pouvez beneficier du compteur et des commandes dérivés de Alt F4 dans vos serveurs sans perdre votre suite actuelle !

Etapes à suivre :

    Ajoutez ALt F4
    Configurez le compteur avec la commande /counter
    Rejoignez le serveur support
    Ouvrez un ticket, informez notre staff du nombre actuel ( hors 0 envoyé précédament par Alt F4)
    Donnez nous l'id du serveur ( voir Etape 0 : Activer le mode developpeur )
    Attendez quelques minutes / heures et c'est parti vous pouvez compter !

Désormais vous etes dans le /top-counter vous pouvez voir votre classement parmis les autres serveurs
